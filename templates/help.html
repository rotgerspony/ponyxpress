Help

Common tasks

  • Save a stop: Map → drop pin → “Save Last Pin”.
  • Delete stops (admin): Map popup “Delete” or Admin → Stops.
  • Scan a parcel: Scanner → Start → point camera → Save (or enable Auto-save).

Troubleshooting

  • Map not loading? Check MAPBOX_TOKEN in .env.
  • Camera blocked? Allow browser camera permission and pick a device.
  • Can’t save? Log in as alice (carrier) or admin.
